# Published results

Only reviewed and redacted evidence bundles belong under `results/published/`.

Do not commit `.approvaltrace/`, raw private captures, API keys, usernames, unrelated prompts, or screenshots containing private information.

Recommended layout:

```text
results/published/<client>/<version>/<scenario>/<run-id>/
```

Every public result must follow `docs/test-protocol.md` and `docs/responsible-disclosure.md`.
